# Nitheesh A — Portfolio

Modern, white & blue themed Java Full Stack Developer portfolio built with **React + Vite**.

## Project Structure

```
nitheesh_portfolio/
├── index.html                          ← Entry (Playfair+Inter fonts, Tabler Icons)
├── package.json
├── vite.config.js
├── README.md
└── src/
    ├── main.jsx                        ← React root
    ├── index.css                       ← Global CSS vars, animations, reveal classes
    └── Protofile_Profile/
        ├── App.jsx                     ← Single page layout (no router)
        ├── App.css                     ← Footer styles
        ├── Nav/
        │   ├── Nav.jsx                 ← Sticky nav, scroll-spy, mobile sidebar
        │   └── Nav.scss
        ├── Home/
        │   ├── Home.jsx                ← Hero + stats
        │   ├── Home.scss
        │   └── nivi.jpeg               ← Profile photo
        ├── About/
        │   ├── About.jsx               ← Bio, internships (Besant + DEV), certs, skills
        │   └── About.less
        ├── Project/
        │   ├── Projects.jsx            ← 7 projects, search + filter
        │   └── Project.css
        └── Contact/
            ├── Contact.jsx             ← Contact form + info
            └── Contact.css
```

## Getting Started

```bash
npm install
npm run dev
```
